
To compile the program you will need:

1.	Microsoft Visual Studio 2010 or greater
2.	Matlab R2012a or greater


Instructions:
1. In file "GRS.c" set the desired mode of the programm ( see doc\CH_INSTR)
2.Build the solution

You can run the programm using VS, for running you need to put your input files to source\approximation directory.

There are some examples of input data for the mode of ellipsoid's approximation for different dimensions in source directory.
All needed sourse code files are in source\code_files directory
