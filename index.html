Ellipsoidal Toolbox for MATLAB

Ellipsoidal Toolbox (ET) is a standalone set of easy-to-use configurable MATLAB routines to perform operations with ellipsoids and hyperplanes of arbitrary dimensions. It computes the external and internal ellipsoidal approximations of geometric (Minkowski) sums and differences of ellipsoids, intersections of ellipsoids and intersections of ellipsoids with halfspaces and polytopes; distances between ellipsoids, between ellipsoids and hyperplanes, between ellipsoids and polytopes; and projections onto given subspaces.

Ellipsoidal methods are used to compute forward and backward reach sets of continuous- and discrete-time piecewise affine systems. Forward and backward reach sets can be also computed for piecewise linear systems with disturbances. It can be verified if computed reach sets intersect with given ellipsoids, hyperplanes, or polytopes.

The toolbox provides efficient plotting routines for ellipsoids, hyperplanes and reach sets.

Required software:

  • **Version 2.0 beta1 **

    • MATLAB 2012a
    • CVX 2.0 build 1010 or higher
    • MPT 3.0.12

  • Version 1.1.3 (or lower)

    • MATLAB 6.5 (or higher)
    • YALMIP (ver. 20120926)

ET version 1.1.3 is also distributed as part of Multi-Parametric Toolbox (MPT)

Third Party Software Packages in ET

ET 2.0

Starting with version 2.0 ET relies on

  • [http://cvxr.com/ CVX] as a more reliable toolbox for solving SDP problems of high dimensionality. [http://cvxr.com/ CVX] - Matlab-based convex modeling framework CVX distribution includes two freeware solvers: SeDuMi (used by default in ET) and SDPT3.
  • [http://control.ee.ethz.ch/~mpt/3/ MPT3] as a toolbox that defines polytope class used in Ellipsoids toolbox.

ET 1.1.3

Legacy versions of ET toolbox (1.1.3 and earlier) use YALMIP+SeDuMi toolbox combination for solving SDP problems.

  • [http://control.ee.ethz.ch/~joloef/yalmip.php YALMIP] - high-level MATLAB toolbox for rapid development of optimization code.
  • [http://sedumi.mcmaster.ca SeDuMi] - MATLAB toolbox for solving optimization problems over symmetric cones.
  • [http://control.ee.ethz.ch/~mpt/2/ MPT2] as a toolbox that defines polytope class used in Ellipsoids toolbox.

These packages are included in the ET distribution. So, you don’t need to download them separately.

Related Software

[http://www-iri.upc.es/people/ros/ellipsoids.html Ellipsoidal Calculus based on Propagation and Fusion]. Propagation is an operation of obtaining an ellipsoid that satisfies an affine relation with given ellipsoid. Fusion is operation of finding an ellipsoid that tightly bounds from outside the intersection of two given ellipsoids.

[http://sysbrain.com/gbt Geometric Bounding Toolbox (GBT)] - commercial MATLAB toolbox that deals with multidimensional convex and nonconvex polytopes and has a limited number of functions operating with ellipsoids.

[http://wiki.grasp.upenn.edu/~graspdoc/wiki/hst/ Hybrid System Tools] - list of free modeling and verification tools for hybrid systems.
